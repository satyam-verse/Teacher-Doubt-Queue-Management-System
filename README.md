# Real-Time Classroom Doubt Queue System

## 📌 Overview
A cloud-based web application that allows students to submit doubts and stores them using AWS.

## 🚀 Features
- Submit doubts in real-time
- Store data in AWS RDS (MySQL)
- Hosted on AWS EC2
- Flask backend

## 🛠️ Tech Stack
- AWS EC2
- AWS RDS (MySQL)
- Python (Flask)

## ⚙️ Setup

pip install -r requirements.txt  
python app.py  

## 🌐 Usage

http://<your-ec2-ip>/add?name=YourName&doubt=YourDoubt
